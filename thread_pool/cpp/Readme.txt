写在最后:
通过仿照以上链接的代码来写这个简单的线程池，解决了我的很多迷惑：
例如1:创建线程后如何给它初始化最初的JOB成员变量，如何让他等待任务的到来。
就是很简单的通过信号量来实现: 线程初始化后进入运行函数，循环查询job成员是否为空，为空则让信号量wait；当job被赋值后，信号量发出信号，然后执行job。
例如2:如何动态的增加或减少线程。

信号量在这里就是起到异步通知的作用，锁的作用是防止多个线程同时修改同一个变量，这两种同步线程的方法都经常用到。
除此之外就是面向对象的设计思想。
